# AurumX Support Page

This directory contains the static support page for AurumX.

## Access

The support page is available at: `/wiki/Support/index.html`

## Hosting Options

You can host this static page using:

1. **GitHub Pages**
   - Enable GitHub Pages in repository settings
   - Select the `main` branch and `/wiki` directory
   - Access at: `https://fxellence-dev.github.io/AurumX/Support/`

2. **Netlify** (recommended for custom domain)
   - Connect your GitHub repository
   - Set build directory to `wiki/Support`
   - Deploy to custom domain like `support.aurumx.app`

3. **Vercel**
   - Import GitHub repository
   - Configure output directory to `wiki/Support`
   - Deploy with automatic SSL

4. **Any Static Host**
   - Simply upload the `index.html` file
   - The page is fully self-contained with inline CSS
